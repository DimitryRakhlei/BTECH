The documentation can be found inside the docs/ folder

./compile.sh compiles the project using cmake

./bin contains the binary used for encryption

./run runs a test of the binary

./encrypt_image.sh encrypts an image provided and displays them using a provided file viewer. 
					~ If you do not have a file viewer just type 'feh' and it will fail at the display step.
					~ Make sure to run ./compile.sh first
					~ Running it without arguments prints usage
					~ ./display folder contains the generated images
